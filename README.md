# Emotion-Sarcasm-Polarity-Analysis
EE6405- Comparative NLP project analyzing emotion polarity (positive vs negative) and sarcasm (sarcastic vs non-sarcastic) in social-media text. Benchmarks TF-IDF + SVM, CNN-LSTM, and DistilBERT models for contextual understanding, with optional AWS SageMaker deployment for cloud-based fine-tuning and inference.
